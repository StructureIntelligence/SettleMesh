paid-tool-api metered

One endpoint that summarizes text and requests delegated payment from the caller's SettleMesh wallet.

Try it



  
Call it yourself

  
Sign in once at /__settle/login, then POST to the endpoint. The session cookie selects
     your SettleMesh wallet as payer; final capture is shown only from trusted platform evidence.

  
curl -X POST https://YOUR-APP.run.settlemesh.io/api/tool \
  -H "Authorization: Bearer <your-settlemesh-session-token>" \
  -H "Idempotency-Key: tool:<one-stable-operation-id>" \
  -H "Content-Type: application/json" \
  -d '{"text":"Long article here...","style":"bullets"}'
